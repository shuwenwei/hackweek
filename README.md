## status:  
0： 成功   
1：失败  
## code：
1 ：以到达最后一页  
2：成功且不是最后一页  
0：没有内容
## /api/token
#### method: POST
#### headers 
```json
{
  "Content-Type": "application/json"
}
```
请求
```json
{
  "username": "aaa",
  "password": "123456"
}
```
响应
```json
{
  "message": "",
  "status": 0
}
```
```json
{
  "message": "",
  "status": 1,
  "token": ""
}
```
